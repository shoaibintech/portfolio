# Shoaib Ali Portfolio

React and Vite portfolio with a GitHub Pages deployment workflow.

## Run locally

```bash
pnpm install
pnpm dev
```

## Deploy with GitHub Actions

1. Push this folder to the root of a GitHub repository.
2. In GitHub, open **Settings → Pages**.
3. Under **Build and deployment**, choose **GitHub Actions** as the source.
4. Push to the `main` branch. The included workflow builds the app and publishes it to GitHub Pages.

The site uses hash based page links, which work correctly on GitHub Pages.
